\# ReactForge 🚀



Crawler desenvolvido em \*\*Node.js\*\* para análise, captura e organização de sites estáticos.



O ReactForge permite acessar uma página web, analisar sua estrutura, coletar informações e salvar os recursos encontrados em uma estrutura local organizada para estudo, análise ou reprodução offline.



\## ✨ Funcionalidades atuais



\- ✅ Acesso automático a páginas web

\- ✅ Download do HTML

\- ✅ Análise da estrutura da página

\- ✅ Extração de:

&#x20; - Título da página

&#x20; - Links encontrados

&#x20; - Arquivos CSS

&#x20; - Scripts JavaScript

&#x20; - Imagens

\- ✅ Organização dos arquivos por projeto

\- ✅ Criação de relatórios em JSON

\- ✅ Sistema inicial de crawler de páginas



\## 🛠️ Tecnologias utilizadas



\- Node.js

\- JavaScript ES Modules

\- Axios

\- Cheerio

\- File System API (Node.js)
